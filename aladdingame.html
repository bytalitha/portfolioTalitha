Aladdin Game

Tijdens dit project heb ik de een vliegende Aladdin geprogrammeerd. Hierbij heb ik gebruik gemaakt van HTML, CSS & Javascript . Door middel van meerdere functies is er een spel gemaakt, waarbij punten gescoord kunnen worden als er een munt of lamp gevangen wordt. Wanneer een rotsblok geraakt wordt, is er een replay optie.
