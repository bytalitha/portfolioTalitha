Portfolio semester 2

Gedurende het tweede semester heb ik al mijn projecten gedocumenteerd in een leerportfolio. Hierbij heb ik het portfolio opgedeeld op basis van mijn leerdoelen, waarbij elk leerdoel de bijbehorende projecten liet zien. Dit portfolio heb ik ontwikkeld met behulp van HTML, CSS en JavaScript.

Note: Project heb ik alleen lokaal beschikbaar

Bekijk op GitHub

Om per leerdoel te kunnen filteren naar een project, heb ik een filterfunctie ontwikkeld met JavaScript. De eerste knop is standaard actief, wat betekent dat deze bij het openen van de pagina geel is en de bijbehorende inhoud weergeeft. Wanneer er op een andere knop wordt geklikt, wordt de actieve status overgedragen en wordt die knop geel, en de bijbehorende inhoud wordt weergegeven. Bovendien heb ik ingesteld dat als er op dezelfde knop wordt geklikt, terug naar de bovenkant van de pagina wordt gescrold.

Design

Voor mijn leerportfolio wilde ik een afbeelding met een passende stijl. Daarom heb ik de afbeelding bewerkt met behulp van Adobe Photoshop, waarbij ik een geel lichteffect heb toegevoegd aan de steen. Bovendien heb ik de achtergrond gewijzigd naar een boslandschap, waardoor de afbeelding een rustigere uitstraling kreeg, en heb ik de gehele afbeelding een donkere tint gegeven.

Als achtergrond voor mijn leerportfolio wilde ik een rookeffect. Mijn naam met rookeffect heb ik gemaakt in Adobe Illustrator. Ik heb zelf een brush ontworpen en deze vervolgens toegepast op mijn naam. Om deze afbeelding als achtergrond te kunnen gebruiken, heb ik de afbeelding ingezoomd op mijn leerportfolio.
